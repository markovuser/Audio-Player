# Audio Player
</div>
<hr>
<div align=center>
  [EN]<br>
<b>Audio Player - audio player on the BASS engine for Windows 10, 11</b><br></br>
Support for a large number of music formats</br>
Internet radio. Listening to radio stations</br>
Hot keys</br>
10-band equalizer and built-in sound effects</br>
Convenient interface</br>
Support for multiple playlists</br>
Search for files by playlists</br>
Tag editor. Support for the main tag formats: ID3v1, ID3v2, APE, Vorbis, WMA.<br><br><hr>

  [RU]<br>
<b>Audio Player — аудиоплеер на движке BASS для Windows 10, 11</b><br></br>
Поддержка большого количества музыкальных форматов</br>
Интернет-радио. Прослушивание радиостанций</br>
Горячие клавиши</br>
10-полосный эквалайзер и встроенные звуковые эффекты</br>
Удобный интерфейс</br>
Поддержка нескольких плейлистов</br>
Поиск файлов по плейлистам</br>
Редактор тегов. Поддержка основных форматов тегов: ID3v1, ID3v2, APE, Vorbis, WMA.<br><br>
